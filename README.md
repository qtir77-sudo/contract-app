# Contract Tool v5.7

## Deploy pe Railway

1. Mergi pe https://railway.app si fa cont (cu GitHub)
2. Click "New Project" → "Deploy from GitHub repo"
3. Selecteaza repo-ul cu aceste fisiere
4. In Settings → Variables, adauga:
   - `ANTHROPIC_API_KEY` = cheia ta API

Gata! Railway porneste automat serverul.
